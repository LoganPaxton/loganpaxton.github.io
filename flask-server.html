How to make a Flask Server (Website)

First, you want to install Python, and the Flask package. Here's how.

LINUX

sudo apt-get install python3

Then run this command.

sudo apt-get pip

To confirm Pip and Python have installed run

pip -v || python --version

If you installed it you should see (maybe not the same as that) pip 24.0 from /nix/store/jnvan6p0bvxvri5k7pp0zrydqvv4gp9i-python3.10-pip-24.0/lib/python3.10/site-packages/pip (python 3.10)

Now, install Flask. If you want to view the package on PyPi, go here

Or, just run pip install Flask
